
## Overview

HomeBites Signup Flow is a web application that simulates the user registration experience for a home-cooked meal marketplace.

The application guides users through a multi-step signup process, allowing them to create an account by entering their personal information, verifying their details, and completing registration. The interface focuses on providing a clean, intuitive, and responsive user experience while demonstrating front-end development best practices for form handling, validation, and navigation between steps.

# HomeBites Signup Flow

This is a code bundle for HomeBites Signup Flow. The original project is available at https://www.figma.com/design/Sp4yKpy5KvypEg0UlQJ5mZ/HomeBites-Signup-Flow.

## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
